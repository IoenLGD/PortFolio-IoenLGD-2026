GLPI 10

Installation sur Debian 12 — Stack LAMP — BTS SIO SISR

Prérequis

  • Debian 12 fraîchement installée (VM ou physique)
  • PHP ≥ 8.2 + extensions : curl, gd, intl, mysql, zip, bcmath, mbstring, xml, bz2
  • MariaDB ≥ 10.6 ou MySQL ≥ 8.0

Installation des paquets

sudo apt update && sudo apt upgrade -y
sudo apt install apache2 php8.2-fpm mariadb-server
sudo apt install php8.2-{curl,gd,intl,mysql,zip,bcmath,mbstring,xml,bz2} -y

Configuration MariaDB

sudo mariadb-secure-installation
sudo mysql -u root -p
CREATE DATABASE db_glpi CHARACTER SET utf8mb4 COLLATE utf8mb4_general_ci;
CREATE USER 'glpi_adm'@'localhost' IDENTIFIED BY 'MotDePasseFort!';
GRANT ALL PRIVILEGES ON db_glpi.* TO 'glpi_adm'@'localhost';
FLUSH PRIVILEGES; EXIT;

Téléchargement GLPI

cd /tmp
wget https://github.com/glpi-project/glpi/releases/download/10.0.16/glpi-10.0.16.tgz
sudo tar -xzvf glpi-10.0.16.tgz -C /var/www/
sudo chown -R www-data:www-data /var/www/glpi

VirtualHost Apache

<VirtualHost *:80>
    ServerName glpi.mon-domaine.fr
    DocumentRoot /var/www/glpi/public
    <Directory /var/www/glpi/public>
        AllowOverride All
        Require all granted
        RewriteEngine On
    </Directory>
    <FilesMatch \.php$>
        SetHandler "proxy:unix:/run/php/php8.2-fpm.sock|fcgi://localhost/"
    </FilesMatch>
</VirtualHost>
sudo a2ensite glpi.conf && sudo a2enmod rewrite proxy_fcgi setenvif
sudo systemctl restart apache2 php8.2-fpm

Finalisation

  • Accéder à http://IP_DU_SERVEUR → suivre l'assistant
  • Connexion par défaut : glpi / glpi → changer immédiatement
  • Supprimer le fichier d'install : sudo rm /var/www/glpi/install/install.php
  • En production : ajouter HTTPS (Let's Encrypt), UFW, backups
