Serveur LAMP

Mise en place sous Debian en environnement virtualisé (VirtualBox) — BTS SIO SISR

Mise à jour du système

sudo apt update
sudo apt upgrade

Confirmer les mises à jour avec 'O' si demandé.

Configuration réseau

  • Vérifier la carte réseau : ip a ou ifconfig
  • VirtualBox : Mode réseau → Accès par pont (bridge) sur la carte physique
  • Si besoin, éditer /etc/network/interfaces puis redémarrer le service réseau
  • Vérifier la connectivité : ping vers routeur et Internet

Accès SSH

sudo apt install openssh-server
sudo adduser userssh

Connexion depuis un client (ex. MobaXterm) : ssh userssh@IP_DU_SERVEUR

Résultat

Le serveur est mis à jour, accessible à distance via SSH et prêt à accueillir des applications web : GLPI, Moodle, WordPress, etc.
