Windows Server

Active Directory

Environnement de test VirtualBox — BTS SIO SISR

Préparation

  • Adresses IP statiques sur serveur et client
  • Désactiver IPv6 et pare-feu (environnement de test)
  • Tester la connectivité Ping entre machines
  • Installer VirtualBox Guest Additions

Installation AD DS + DNS

  • Ajouter le rôle "Active Directory Domain Services" via le Gestionnaire de serveur
  • Promouvoir en contrôleur de domaine → Nouvelle forêt (ex. wsrv.labo)
  • Créer un compte admin dédié (ex. admin_AD)

Utilisateurs et rattachement

  • Créer un utilisateur test (ex. BTSSIO / Passw0rd)
  • Rattacher le poste client au domaine → Redémarrer

Structure OU, Groupes, GPO

  • Créer OU société → sous-OU par service
  • Groupes : Sécurité / Distribution (Global, Universel…)
  • GPO : mappage lecteurs, fond d'écran, redirection dossiers

Profils et scripts

  • Profils itinérants / obligatoires
  • Scripts de connexion (.bat) dans SYSVOL\Scripts

Bonnes pratiques

  • Politique mots de passe stricte
  • Désactiver les comptes inactifs
  • Surveiller les journaux + gpupdate /force
